# Security Policy

## 🔒 安全聲明

本項目是一個**教育演示項目**，用於展示區塊鏈安全問題。所有智能合約均部署在 Sepolia 測試網，不涉及真實資金。

## 🚨 重要安全提醒

### 私鑰安全
- ⚠️ **切勿**將真實錢包的私鑰用於本項目
- ⚠️ **切勿**將含有真實資金的私鑰提交到版本控制
- ✅ 請為測試網專門創建新的錢包地址
- ✅ 從 [Sepolia Faucet](https://faucet.sepolia.dev/) 獲取免費測試 ETH

### API 密鑰安全
- 所有 API 密鑰（Etherscan、RPC 等）應保存在 `.env` 文件中
- `.env` 文件已在 `.gitignore` 中排除，請勿提交到 Git
- 如果意外提交了敏感信息，請立即：
  1. 輪換所有密鑰
  2. 使用 `git filter-branch` 或 BFG Repo-Cleaner 清理歷史記錄

### 部署前檢查清單
- [ ] 確認使用測試網錢包
- [ ] 確認 `.env` 未被提交
- [ ] 檢查 Git 歷史無敏感信息：`git log -p .env`
- [ ] 所有 API 密鑰使用環境變數

## 🐛 漏洞報告

如果您發現本項目中的安全問題，請通過以下方式報告：

1. **優先方式**：在 GitHub 上創建 [Security Advisory](https://github.com/ImL1s/erc20-honeypot-demo/security/advisories)
2. **備選方式**：發送郵件至項目維護者（請在 GitHub profile 中查找聯繫方式）

請**勿**在公開 Issue 中披露安全漏洞。
